    1. What is Semantic HTML?
	- Semantic HTML introduces meaning to the text rather than just presentation. For example <h1> tag lets the computer and the programmer know that that certain text is a heading.
    2. What is HTML used for? 
	- HTML is used for the backbone of a website. All the content is displayed using HTML.
    3. What is an attribute and where do we put it? 
	- Attributes provide additional information about HTML elements and they go inside the opening tags.
    4. What is the h1 tag used for? How many times should I use it on a page?
	-h1 tag is used for the most important title/heading. You can use it as many times as you like, but just remember that it'll loose meaning if overused.
    5. Name two tags that have required attributes
	- the a tag and img tag.
    6. What do we put in the head of our HTML document? 
	-heading
    7. What is an id? 
	- id is an attribute unique to a single tag. 
    8. What elements can I add an id to? 
	- you can only add an id attribute to any html element as long as they have different ids.
    9. How many times can I use the same id on a page? 
	-only once
    10. What is a class? 
	-classes group elements so that they can be manipulated in css.
    11. What elements can I add a class to? 
	- any tags.
    12. How many times can I use the same class on a page? 
	-as often as needed
    13. How do I get my link to open in a new tab?
	- add a "_blank" attribute
    14. What is the alt attribute used for? 
	- alt attribute is used for when the text cannot appear or for making an image accessible to the visually impaired
    15. How do I reference an id?
	- #id
    16. What is the difference between a section and a div
	- <section> divides the material inside it and therefore has meaning while <div> is presentation
    17. What is CSS used for? 
	- styling the webpage like design the structure of a house.
    18. How to we select an element? Example - every h2 on the page
	-a tag can be selected by calling it as it is
    19. What is the difference between a class and an id? - Give me an example of when I might use each one
	-multiple elements can belong to a class but only one can have the same id
	-if you want all of the different heading to have the same font-color.. you can achieve this by calling their class
	-if you want a specific word to have a different position.. this would best be done using its id
    20. How do we select classes in CSS?
	- .class
    21. How do we select a p element with a single class of “human””?
	- .human p{}
    22. What is a parent child selector? When would this be useful? 
	- a selector that has more specificity than a parent. so the child selector is nested inside the parent, giving the child selector ability to manipulated only its elements. 
    23. How do you select all links within a div with the class of sidebar?
	- Since all the links are nested inside the class of sidebar, just call the selector class of sidebar.
    24. What is a pseudo selector?
	- selects elements only when they're in a specific state such as ":hover".
    25. What do we use the change the spacing between lines?
	- use line-height property
    26. What do we use to change the spacing between letters?
	- letter-spacing:
    27. What do we use to to change everything to CAPITALS? lowercase? Capitalize?
	- text-transform: capitals, lowercase, capitalize;
    28. How do I add a 1px border around my div that is dotted and black?
	- div{
		border: 1px dotted black;
		}
    29. How do I select everything on the page? 
	- *{}
    30. How do I write a comment in CSS?
	- /**/
